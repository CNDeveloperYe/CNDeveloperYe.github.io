下拉选择

发表于 2016-12-02

###需求描述

  • 点击某一行时，在该行的下方弹出一个pickView，选择完成之后，再次点击该行收回.

###做法

  • 1.将列表中左边显示的文字都提取到一个titleArr中,同时定义展开的标记的字符串代码

  • 2.点击某一行时，需要判断当前的titleArr是否存在一个展开的标记的字符串代码 

    存在：说明有展开，获取该标记的位置，tableView中删除该位置的cell，数组中也同时移除。
不存在：数组中根据当前点击的位置+1，然后添加一个展开的标记的字符串，tableView中添加该位置的cell

  • 3.cellForRowAtindexPath的方法中，titleArr数组中依次展示cell，如果有展开的标记的字符串代码

    根据展开的标记的字符串的位置-1，获取需要设置 选择文字的cell，如果该cell没有文字则给一个默认的文字。
当展开cell选择完成之后，将选择的结果赋值给相应的cell
  • 4 高度的调整代码
  • 5 效果图效果

####代码1

let enpandingMark = "expend"

var nameArr = ["我","是","帅","哥","！"]

var titleArr = ["我帅","我比较帅","我比别人帅","别人没我帅"]

####代码2

func tableView(_ tableView: UITableView, didSelectRowAt indexPath: IndexPath) {

    //检测当前有没有已经展开的
    let isEnpending = self.nameArr.contains(enpandingMark)

    if isEnpending {//如果展开,则回收

        //获取之前展开的位置
        let index = self.nameArr.index(of: enpandingMark)!
        //关闭掉
        nameArr.remove(at: index)

        let indexPath = IndexPath(row: index, section: 0)
        //删除某一个cell
        tableView.deleteRows(at: [indexPath], with: UITableViewRowAnimation.none)

    }else{

        //没有展开cell
        nameArr.insert(enpandingMark, at: indexPath.row + 1)

        let indexPath = IndexPath(row: indexPath.row + 1, section: 0)

        //添加某一个新的cell
        tableView.insertRows(at: [indexPath], with: UITableViewRowAnimation.none)

    }

}

####代码3

   func tableView(_ tableView: UITableView, cellForRowAt indexPath: IndexPath) -> UITableViewCell {

    var cell =  tableView.dequeueReusableCell(withIdentifier: "reuse")

    if cell == nil {

       cell = UITableViewCell(style: UITableViewCellStyle.value1, reuseIdentifier: "reuse")
    }

    cell?.accessoryType = UITableViewCellAccessoryType.disclosureIndicator

    cell?.textLabel?.text = nameArr[indexPath.row]

    //有展开的
    if self.nameArr[indexPath.row] ==  enpandingMark{

        //获取之前展开的位置
        let index = self.nameArr.index(of: enpandingMark)!

        let indexPath = IndexPath(row: index - 1, section: 0)

        //获取需要改变标题的cell
        let changeCell = tableView.cellForRow(at: indexPath)

        //第一次给个默认值
        if changeCell?.detailTextLabel?.text == nil {

            changeCell?.detailTextLabel?.text = titleArr.first
        }

        //根据需要改变标题的cell里面的文字来旋转CustChooseCell 转到那个位置
        let title = changeCell?.detailTextLabel?.text ?? titleArr.first!

        let selectIndex = titleArr.index(of: title)

        cell = Bundle.main.loadNibNamed("CustChooseCell", owner: nil, options: nil)?.last as! CustChooseCell

        let custChooseCell = cell as! CustChooseCell

        custChooseCell.titleArr = titleArr

        //选中标题中已经选择的位置
        custChooseCell.pickerView.selectRow(selectIndex!, inComponent: 0, animated: false)

        custChooseCell.chooseTheTitle = { title in

            changeCell?.detailTextLabel?.text = title
        }
    }

    return cell!
}

func tableView(_ tableView: UITableView, heightForRowAt indexPath: IndexPath) -> CGFloat {

    let title = self.nameArr[indexPath.row]

    if title == enpandingMark{

        return 80
    }

    return 44
}

URL和Path的区别和联系

相同:都是表示资源的路径

URL:第一部分必须为协议名字

1.http／https：超文本传输协议
2.file:本地或者网上分享的文件(局域网)

filePath:默认的采用了file协议，转化为URL时可以采用手动或者系统的方法加上 “file://“

  • 写法一

    let ulrString = "http://image.tianjimedia.com/uploadImages/upload/20161030/ybblbgljviljpg.jpg"
let url = URL(string: ulrString)

  • 写法二

    let localPathString = "file://" + "/Users/yeguixian/YeGuixianBlog/source/_posts/Unknown.jpg"
 let url = URL(string: localPathString)

  • 写法三

    let localPathString = "/Users/yeguixian/YeGuixianBlog/source/_posts/Unknown.jpg"
 let url = URL(fileURLWithPath: localPathString)
